Mode d'emploi

  1. Passe ton premier tour sans bouger afin d'afficher les déplacements prévus du minogolem.
  2. Entre les 8 prochains déplacements du minogolem. Clic droit sur le minogolem, « Afficher les enchantements », puis passe ta souris sur les deux enchantements.
  3. Clique sur Play pour recevoir 4 actions.
  4. Exécute les 4 instructions.
  5. Entre les 4 prochains déplacements du minogolem. Passe ta souris sur l'enchantement "Labyrinthe" et note les 4 derniers états.
  6. Répète les étapes 3 à 5 jusqu'à la victoire.
*Victoire non garantie, le combat est par nature aléatoire. 90% de taux de réussite lors des test*
